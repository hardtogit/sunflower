### 时光之穴--

> 时光之穴是网络游戏《魔兽世界》中的一个地名，位于卡利姆多大陆南端的塔纳利斯。
该地区是一个副本群，内有八个副本，这八个副本分别记录了魔兽的关键历史 
以此来表示组件的核心模块

## 安装

- npm
```js
npm set registry http://39.100.136.61:9001

npm i @cot/core
npm i @cot/ui
npm i @cot/wx
...按照需要安装
```

- yarn
```js
yarn config set registry http://39.100.136.61:9001
yarn add @cot/core
yarn add @cot/ui
yarn add @cot/wx
```
