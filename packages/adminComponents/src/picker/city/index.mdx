# CityPicker

## 使用方法

```js
import 'antd/dist/antd.css'
import City from './';
<City></City>

```
- 基本使用
<Row>
    <Col span={6}>
<City></City>
</Col>
</Row>



- 多选

<Row>
    <Col span={6}>
     <City multiple regionType='city'></City>
    </Col>
</Row>

```js
     <City multiple></City>
```

- 禁用
<Row>
    <Col span={6}>
     <City disabled></City>
    </Col>
</Row>

```js
     <City disabled></City>
```
- regionType

## API

|名称|默认值|是否必须|说明|
|----|  ----| ----| ----|
|multiple|false|否|是否多选|
|disabled|false|否|是否禁用|
|regionType|'area'|否|支持 省province  市city  区area|
|simple|true|否| true返回id或id组成的数组，false返回区域对象或区域对象组成的数组|
